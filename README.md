## 🚀 What this is

npm-kairos helps you turn time values into human-friendly text in JavaScript and TypeScript. It is built for cases where `86400000` should show as `1 day`, not as a long number.

Use it when you want to work with time in a way that is easier to read in your app, script, or project. It fits well in tools that show time, date ranges, wait periods, or schedule text.

Use npm-kairos when you want to show time in a human way.

Examples of what that means:

- `3600` becomes `1 hour`
- `86400000` becomes `1 day`
- larger values can become longer time spans that are easier to read

It is useful for:
